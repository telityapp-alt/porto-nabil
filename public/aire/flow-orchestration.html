Agentic Orchestration
1 Human + 24+ Specialist Agents: Slack-like Workspace untuk HR Work
Orchestrator
Aire (Main Agent)
Router & Coordinator
Central orchestrator yang receive user goals, decompose jadi subtasks, route ke specialist agents, dan merge results. Bukan monolithic chatbot — ini adalah traffic controller untuk 24+ specialist agents.
Core Capabilities
  • Intent detection (hiring task, onboarding, compliance, review)
  • Task decomposition ("12 new joiners" → contracts + onboarding + comms)
  • Agent routing (assign subtasks ke specialist agents)
  • Result merging (compile specialist outputs jadi actionable plan)
  • Context maintenance (track state across multi-turn conversations)

How Orchestration Works: User Goal → Multi-Agent Execution
1
User Input: Natural Language Goal
User type satu line di Aire workspace: "@aire we have 12 new joiners next Monday. Stand up onboarding, contracts, comms, and a day-7 check." No structured forms, no dropdown menus.
2
Aire (Orchestrator): Intent Detection
Aire parse intent: onboarding task, 12 employees, 4 subtasks (onboarding plan + contracts + comms + check-in). Identify which specialist agents needed.
3
Task Routing: Parallel Agent Dispatch
Aire route ke 4 specialist agents in parallel: Onboarding Agent (build plan), Compliance Agent (generate contracts), Brand Voice Agent (draft comms), Onboarding Agent lagi (schedule day-7 check). Semua bekerja simultaneously.
4
Specialist Execution: Context-Aware Work
Each agent access relevant context (role templates, company policies, employee data) dan execute subtask. Onboarding Agent pull role-specific checklist, Compliance Agent inject employee names into contract template, etc.
5
Result Merging: Unified Deliverable
Aire collect outputs dari 4 agents, merge jadi satu cohesive plan: onboarding checklist (12 employees), contract PDFs (ready untuk esign), welcome emails (drafted), day-7 calendar invites (scheduled). User review one unified deliverable, not 4 separate outputs.
6
Human Review & Approval
User review merged output di morning. Approve, edit, atau reject specific pieces. Human always in the loop — agents propose, human decides.
