Core Psychology Frameworks
BJ Fogg's Behavior Model
Dr. BJ Fogg, Stanford
B = MAT (Behavior = Motivation × Ability × Trigger). Focus pada reduce ability barriers (bikin learning lebih mudah) dan increase trigger effectiveness.
Hooked Model
Nir Eyal
Trigger → Action → Variable Reward → Investment. User invest waktu/data, creating commitment yang bawa mereka balik lagi.
Flow State Theory
Mihaly Csikszentmihalyi
Optimal challenge-skill balance. Questions adapt difficulty buat keep user di flow zone — not too hard (anxiety), not too easy (boredom).
Loss Aversion
Kahneman & Tversky
Orang lebih takut losing daripada gaining. Streaks leverage ini: user ga mau break progress mereka, driving daily returns.
Operant Conditioning
B.F. Skinner
Variable ratio reinforcement schedules (unpredictable rewards) create strongest behavior patterns. XP bonuses trigger dopamine release.
Self-Determination Theory
Deci & Ryan
Intrinsic motivation butuh autonomy (user choose path), competence (clear progress), dan relatedness (social proof via leaderboards).
Key Insights dari Research
  • Streaks adalah retention driver terkuat — 67% daily active users punya streak >7 hari (data Duolingo)
  • Variable rewards outperform fixed rewards 3:1 — unpredictable XP bonuses increase engagement 40%
  • Social proof reduce anxiety di high-stakes context — liat progress peers normalize struggle dan build credibility
  • Immediate feedback loops itu critical — waiting >2 detik untuk answer validation break flow dan reduce completion
  • Progressive mastery prevent drop-off — adaptive difficulty keep 85% user di optimal challenge zone vs 45% dengan fixed difficulty
  • Endowed progress effect boost conversion — pre-seeded progress bars increase completion rate 25% (Nunes & Drèze, 2006)
