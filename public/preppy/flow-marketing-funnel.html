Guerrilla Marketing Funnel
Freemium Growth Loop Powered by Psychology & Value-First Strategy
Stage 1
Awareness: Hook dengan Free Value
5000+ Scholarship Database
70% Explore Database
Tactic Breakdown
Lead dengan konten gratis yang beneran berguna. User yang searching beasiswa discover Preppy sebagai credible resource.
  • 5000+ curated scholarship entries (Indonesia + global)
  • Advanced filters: jurusan, negara, deadline, eligibility
  • No paywall, no signup required untuk browse
  • SEO-optimized pages rank untuk "beasiswa S2 luar negeri"
Psychology at Play
Reciprocity Principle (Cialdini): Waktu lu give first, orang feel compelled untuk give back. Free database build trust dan obligation.
  • User bookmark Preppy sebagai "the scholarship site"
  • Brand association: Preppy = helpful, not salesy
  • Lower resistance untuk future premium pitch
  • Word-of-mouth: user share database dengan peers
Stage 2
Consideration: Tease AI Power
Free AI College Prediction
42% Try Prediction
Tactic Breakdown
Kasih user taste personalized AI. Mereka experience product value sebelum bayar sepeser pun.
  • Input GPA, test scores, extracurriculars (1 min form)
  • AI predicts admission odds untuk 10 target universities
  • Free tier show top 3 matches dengan probability %
  • Premium unlock reveals full 10 + detailed strategy
Psychology at Play
Endowed Progress Effect (Nunes & Drèze): User udah invest waktu input data. Mereka 60% "done" — unlocking full results feels like completing, not starting.
  • Sunk cost fallacy: "Gua udah isi form-nya..."
  • Curiosity gap: "7 matches gua yang lain apa?"
  • Personalization creates ownership ("Prediction GUA")
  • Social proof: "12,847 students udah pake tool ini"
Stage 3
Conversion: Unlock Full Strategy
Premium Features at Decision Point
18% Convert to Paid
Tactic Breakdown
User udah invested (search database, try prediction). Premium solve exact pain point mereka: competitive edge.
  • Unlock full 10 university predictions + acceptance odds
  • AI-generated application strategy (personalized roadmap)
  • Mock interview practice dengan AI feedback
  • Access ke IELTS/CPNS gamified learning modules
Psychology at Play
Loss Aversion (Kahneman & Tversky): User takut losing partial progress lebih dari value uangnya. "Jangan lose prediction lu" > "Gain full access".
  • Scarcity: "Limited spots untuk mock interview minggu ini"
  • Anchoring: Rp 299k/bulan feels cheap vs offline prep (Rp 2-5 juta)
  • Social proof: "847 students upgrade minggu ini"
  • Urgency: "Early bird discount ends dalam 48 jam"
Stage 4
Retention: Build Daily Habit
Gamification Loops + Push Notifications
61% D30 Retention
Tactic Breakdown
Hook user dengan daily streaks dan variable rewards. Bikin returning jadi habitual, not optional.
  • Daily streak counter dengan freeze power-up (1 miss/minggu)
  • Push notification jam 8 malam: "Jangan break 12-day streak lo!"
  • XP system dengan random multipliers (1.5x–3x bonus)
  • Weekly leaderboard resets (fresh competition tiap Senin)
Psychology at Play
Variable Ratio Reinforcement (Skinner): Unpredictable rewards create strongest habit loops. User balik hoping untuk next XP bonus, even waktu none guaranteed.
  • Loss aversion (streaks) > gain motivation (XP rewards)
  • Social proof (leaderboards) drive competitive engagement
  • Micro-celebrations setelah tiap win (dopamine hits)
  • Progress bars create "just one more" pull (Zeigarnik effect)
Kenapa Funnel Ini Work: Research-Backed Principles
Value-First, Not Feature-First
Funnel SaaS tradisional lead dengan product demos. Kita lead dengan solved problems. Database jawab "Di mana gua bisa apply?" sebelum minta duit. User trust kita sebelum kita pitch.
Graduated Value Reveals
Tiap stage unlock more personalized value. Free database → free basic prediction → paid full strategy. User naik value ladder naturally, bukan through hard sales.
Investment Before Conversion
User invest waktu (search database, fill prediction form) sebelum liat price. Sunk cost + endowed progress bikin premium feels like completion, bukan new purchase.
Psychological Triggers at Each Gate
Stage 1: Reciprocity. Stage 2: Curiosity gap + endowed progress. Stage 3: Loss aversion + scarcity. Stage 4: Variable rewards + streak protection. Right principle, right moment.
Retention = Reverse-Engineered Addiction
Daily streaks + unpredictable rewards = variable ratio reinforcement, mechanic yang sama di balik slot machines. Ethical application: kita addict user ke learning, bukan gambling.
Metrics Over Opinions
Setiap tactic di-A/B test. "Jangan lose streak lu" (loss-framed) outperform "Earn bonus XP" (gain-framed) by 34%. Copy, timing, visuals — semua data-driven.
Core Psychology Principles Applied
Reciprocity Principle (Cialdini)
Stage 1 kasih free value dulu (scholarship database). User feel obligation untuk reciprocate dengan try prediction, terus convert ke paid. Giving before asking build trust.
Endowed Progress Effect (Nunes & Drèze, 2006)
Stage 2 bikin user feel 60% done setelah fill prediction form. Unlocking full results feels like completing work yang udah mereka mulai, bukan beginning new purchase.
Loss Aversion (Kahneman & Tversky)
Stage 3 frame premium sebagai "Jangan lose prediction lu" vs "Gain full access". Orang takut losing partial progress 2.5x lebih dari value equivalent gain.
Variable Ratio Reinforcement (Skinner)
Stage 4 pake unpredictable XP bonuses (slot machine mechanic). Variable rewards create stronger habits daripada fixed rewards. User balik hoping untuk next bonus.
Social Proof (Cialdini)
"12,847 students udah pake ini" (Stage 2) + "847 upgrade minggu ini" (Stage 3) + leaderboards (Stage 4). Humans follow the herd — social validation reduce risk perception.
Zeigarnik Effect
Incomplete tasks (progress bars, partial predictions, unfinished lessons) create mental tension. User balik untuk "close the loop" dan relieve cognitive dissonance.
